# HazardX

HazardX is a machine learning based workplace safety prediction platform designed to analyze historical safety data and identify potential accident risks before they occur.

The project combines a React frontend with a Python-based machine learning backend to provide risk predictions, safety insights, and accident severity analysis.

## Features

- Workplace accident risk prediction
- Machine learning based severity analysis
- Modern React + Vite frontend
- Responsive UI built with Tailwind CSS
- Fast API integration support
- Data-driven safety insights
- Real-time prediction workflow

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Lucide React

### Backend
- Python
- FastAPI / Flask
- Scikit-learn

### Machine Learning
- Logistic Regression (baseline)
- Random Forest Classifier (primary)
